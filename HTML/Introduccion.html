Que es HTML?

HTML significa lenguaje de marcado de hipertexto. Es un lenguaje de marcado utilizado para crear páginas web. HTML es un lenguaje basado en texto, lo que significa que se compone de texto y etiquetas. Las etiquetas se utilizan para identificar diferentes elementos de una página web, como encabezados, párrafos, imágenes y enlaces. HTML es un lenguaje muy simple de aprender. Puede comenzar a crear páginas web básicas con solo unas pocas etiquetas básicas. A medida que se familiarice con HTML, podrá aprender técnicas más avanzadas, como la creación de tablas, listas y formularios. HTML funciona mediante el uso de etiquetas. Cuando un navegador web carga una página web, interpreta las etiquetas HTML y muestra la página en consecuencia. El navegador representará el texto, las imágenes y otros elementos de la página según lo especificado por las etiquetas HTML. HTML es la base del desarrollo web. Si quieres crear páginas web, necesitas aprender HTML. Estos son algunos de los beneficios de aprender HTML:


  • HTML es un lenguaje versátil que se puede utilizar para crear una amplia variedad de páginas web.
  • HTML es un lenguaje gratuito y de código abierto, lo que significa que hay muchos recursos disponibles para ayudarte a aprenderlo.
  • HTML es un lenguaje relativamente fácil de aprender, incluso para principiantes.
  • HTML es la base del desarrollo web, por lo que aprender HTML es una excelente manera de comenzar en este campo

Breve historia de HTML

Su historia se remonta a los primeros días de la web. Aquí hay una breve historia de HTML: 1. Orígenes: En 1989, Tim Berners-Lee, científico de la computación británico, desarrolló el concepto de la World Wide Web mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear) en Suiza. Berners-Lee creó HTML como un lenguaje para estructurar y vincular documentos en un formato que pudiera ser compartido y accedido a través de Internet. 2. HTML 1.0: La primera especificación oficial de HTML, conocida como HTML 1.0, se lanzó en 1993. Era un lenguaje básico que permitía la estructuración de texto, la creación de enlaces y la inclusión de imágenes en las páginas web. 3. HTML 2.0: En 1995, se lanzó HTML 2.0, que introdujo nuevas características como la capacidad de incluir formularios interactivos y soporte para tablas. Esta versión fue ampliamente adoptada y sentó las bases para el crecimiento explosivo de la web. 4. HTML 3.2 y 4.0: En 1997, se lanzó HTML 3.2, que agregó más elementos y atributos al lenguaje. Luego, en 1999, HTML 4.0 se lanzó con mejoras significativas, como soporte para hojas de estilo CSS y el concepto de iframes (marcos internos). HTML 4.0 estableció un conjunto de características ampliamente utilizado durante muchos años. 5. XHTML: En 2000, el Consorcio World Wide Web (W3C) introdujo XHTML (Extensible HyperText Markup Language), que combinaba HTML con la sintaxis XML (Extensible Markup Language). XHTML siguió las reglas más estrictas de XML y promovió una estructura más coherente y limpia en las páginas web. 6. HTML5: La versión más reciente y significativa de HTML es HTML5, que se lanzó en 2014. HTML5 introdujo nuevas etiquetas semánticas, elementos de multimedia integrados (como audio y video), soporte para gráficos vectoriales (SVG) y capacidades de almacenamiento local. También mejoró la accesibilidad y proporcionó una mejor integración con CSS y JavaScript. HTML5 se ha convertido en el estándar actual para el desarrollo web, y ha permitido la creación de sitios web más ricos y dinámicos con mayor facilidad. A lo largo de los años, HTML ha evolucionado para adaptarse a las necesidades cambiantes de la web y seguirá evolucionando en el futuro. Es el lenguaje clave que ha permitido la expansión y el impacto global de Internet tal como lo conocemos hoy.

Conceptos fundamentales:

  1. Elemento HTML: es un objeto dentro del documento de HTML que consiste en una etiqueta inicial, un contenido, y una etiqueta final. Ejemplo: <h1>Titilo 1</h1>
  2. Etiqueta HTML: es el nombre de un elemento de HTML dentro de paréntesis angulares. Ejemplos <h1>, <p>, <li>, etc.
  3. Atributo HTML: es una configuración adicional para un elemento de HTML. Los atributos se ubican en la etiqueta de apertura de un elemento. Ejemplo: <a href=""> </a>, donde href es un atributo del elemento <a> </a>.
  4. Selector: un selector es el recurso que HTML provee para poder seleccionar elementos específicos dentro de un documento HTML. Las etiquetas, y los atributos class e id, son los selectores principales.
  5. Clase: bajo el contexto de HTML, una clase (class) es un atributo que permite identificar uno a varios elementos de una página de HTML, a los que eventualmente se les quieren aplicar estilos.
  6. id: es el atributo que permite asociar un identificador único a un elemento específico del documento HTML. Ningún atributo de tipo id debe tener el mismo valor (mismo id) en las misma página.

HTML 5

HTML5 es la quinta y última revisión importante del estándar HTML. Fue publicado en 2014 y es la versión de HTML más utilizada. HTML5 presenta una serie de características nuevas, que incluyen:

  • Nuevos elementos: HTML5 introduce una serie de nuevos elementos, como header , footer y article. Estos elementos se pueden utilizar para marcar la estructura de una página web de una forma más semántica.
  • Nuevos atributos: HTML5 introduce una serie de nuevos atributos, como el atributo srcset para imágenes. Este atributo le permite especificar varias fuentes de imágenes para diferentes tamaños de pantalla, lo que puede mejorar el rendimiento de sus páginas web.
  • Nuevas formas de aplicar estilo a su HTML: HTML5 presenta varias formas nuevas de aplicar estilo a su HTML, como el elemento style y el atributo scoped. Estas características pueden ayudarlo a crear estilos más consistentes y fáciles de mantener para sus páginas web.

    • HTML5 es un lenguaje poderoso que se puede usar para crear páginas web complejas e interactivas. Al comprender las nuevas características de HTML5, puede crear páginas web que sean tanto informativas como visualmente atractivas.

    Estos son algunos de los beneficios de usar HTML5:

  • Marcado más semántico: HTML5 introduce una serie de elementos nuevos que se pueden utilizar para marcar la estructura de una página web de una forma más semántica. Esto hace que sus páginas web sean más accesibles para los motores de búsqueda y otro software, y también puede hacer que su código sea más fácil de leer y mantener.
  • Rendimiento mejorado: HTML5 introduce una serie de características nuevas que pueden mejorar el rendimiento de sus páginas web, como el atributo srcset para imágenes. Este atributo le permite especificar varias fuentes de imágenes para diferentes tamaños de pantalla, lo que puede ayudar a mejorar el tiempo de carga de sus páginas.
  • Mayor compatibilidad: HTML5 es compatible con todos los principales navegadores web, incluidos Chrome, Firefox, Safari y Edge. Esto significa que sus páginas web serán accesibles a una amplia gama de usuarios.
